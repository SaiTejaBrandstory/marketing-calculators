Local vs. Global Branding Performance Calculator

This tool helps compare the branding performance of your business at local and global levels. It takes input from different regions and returns the score based on the local and global market performance.

Formula:

Brand Performance Score = (Local Branding Score + Global Branding Score) / 2

Example Calculation:

Local Branding Score: 70

Global Branding Score: 85

Brand Performance Score = (70 + 85) / 2 = 77.50
