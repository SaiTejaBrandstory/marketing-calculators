Public Relations (PR) ROI Calculator

The PR ROI Calculator helps determine the effectiveness of your public relations campaigns by calculating the return on investment based on the media value, campaign cost, and other key factors.

Formula:

PR ROI = (Media Value - Campaign Cost) / Campaign Cost

Example Calculation:

Media Value: ₹1,50,000

Campaign Cost: ₹50,000

PR ROI = (₹1,50,000 - ₹50,000) / ₹50,000 = 2.00

Media Value (₹)
