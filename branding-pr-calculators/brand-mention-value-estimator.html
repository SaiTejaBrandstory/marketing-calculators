Brand Mention Value Estimator

The Brand Mention Value Estimator helps estimate the monetary value of mentions your brand receives across different platforms like news, blogs, social media, etc.

Formula:

Brand Mention Value = (Social Media Mention Value + News Mention Value + Blog Mention Value) / 3

Example Calculation:

Social Media Mention Value: ₹5000

News Mention Value: ₹7000

Blog Mention Value: ₹6000

Brand Mention Value = (5000 + 7000 + 6000) / 3 = ₹6000
